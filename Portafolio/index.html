Juego del Ahorcado

La aplicación web consiste en presentarle al usuario una interfaz donde puede dar inicio y resetear el juego cuando lo desee. Al iniciar la partida el sistema por debajo elije al azar una palabra oculta y el jugador deberá teclear las letras para averiguar dicha palabra. Si acierta todas las letras el sistema le avisará de su victoria, caso contrario, le avisará que perdió la partida y tendrá que comenzar otra.
